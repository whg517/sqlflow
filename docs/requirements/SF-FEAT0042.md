## 验收标准

1. 深色模式下所有页面无明显视觉瑕疵
2. 颜色对比度符合 WCAG AA 标准（≥ 4.5:1）
3. 边框、背景、文字在深色模式下清晰可辨
4. 不影响浅色主题显示
5. CSS 变量使用一致，无硬编码颜色值
